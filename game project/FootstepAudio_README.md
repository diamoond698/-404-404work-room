# 脚步声管理功能（精简版）

## 功能说明
实现人物走动时播放脚步声音效，停下时停止音效的精简功能。

## 实现步骤

### 1. 准备音频文件
1. 准备一个脚步声音频文件（WAV、MP3、OGG等格式）
2. 将音频文件复制到Unity项目的 `Assets` 文件夹中

### 2. 添加脚本
1. 将 `FootstepAudio.cs` 脚本复制到Unity项目的 `Assets` 文件夹中
2. 将脚本附加到角色对象上

### 3. 配置脚本参数
1. 选中角色对象
2. 在Inspector面板中找到 `FootstepAudio` 组件
3. 配置以下参数：

   **音频设置**：
   - `Footstep Sound`：拖拽脚步声音频文件到该字段
   - `Audio Source`：如果角色已有AudioSource组件，拖拽到该字段；否则会自动创建

   **移动设置**：
   - `Move Threshold`：移动速度阈值（默认0.1）
   - `Step Interval`：脚步声播放间隔（默认0.5秒）

### 4. 测试功能
1. 运行游戏
2. 控制角色移动，观察是否播放脚步声
3. 停止移动，观察音效是否停止
4. 调整 `Step Interval` 参数，改变脚步声播放频率

## 脚本说明

### FootstepAudio.cs（精简版）
- 使用Rigidbody2D的速度检测角色是否移动
- 移动时按设定间隔播放脚步声
- 停止时自动停止脚步声
- 自动添加AudioSource组件（如果没有）

## 参数说明

### Move Threshold（移动阈值）
- **作用**：判断角色是否移动的速度阈值
- **默认值**：0.1
- **调整建议**：根据需要调整灵敏度

### Step Interval（脚步间隔）
- **作用**：脚步声播放的时间间隔
- **默认值**：0.5秒
- **调整建议**：根据角色移动速度调整

## 注意事项
- 确保角色有Rigidbody2D组件
- 只支持单个脚步声音频文件
- 移动时自动播放，停止时自动停止
