# 角色陷阱死亡与复活功能

## 功能说明
实现当角色碰到标签为"traps"的对象时死亡，播放死亡动画，并在死亡后复活回到名为"门"的对象附近的功能。

## 实现步骤

### 1. 准备工作
1. **标签设置**：
   - 为陷阱对象设置Tag为"traps"
   - 为门对象设置Tag为"门"

2. **动画准备**：
   - 确保角色有Animator组件
   - 在动画控制器中添加"Death"和"Respawn"触发器参数
   - 创建死亡动画和复活动画

3. **移动脚本**：
   - 确保角色已附加`EightDirectionMovement`脚本
   - 如果使用其他移动脚本，需要修改`Die()`方法中禁用移动的代码

### 2. 添加脚本
1. 将 `PlayerTrapDeath.cs` 脚本复制到Unity项目的 `Assets` 文件夹中
2. 将脚本附加到角色对象上

### 3. 配置脚本参数
1. 选中角色对象
2. 在Inspector面板中找到 `PlayerTrapDeath` 组件
3. 调整以下参数：
   - `Trap Tag`：陷阱对象的标签（默认"traps"）
   - `Door Tag`：门对象的标签（默认"门"）
   - `Respawn Delay`：死亡后到复活的延迟时间（默认1.5秒）

### 4. 测试功能
1. 运行游戏
2. 控制角色接触陷阱对象，观察是否播放死亡动画
3. 等待延迟时间后，观察角色是否在门附近复活
4. 测试多次死亡和复活，确保功能正常

## 脚本说明

### PlayerTrapDeath.cs
- 检测角色与陷阱对象的碰撞
- 播放死亡动画
- 禁用角色移动
- 延迟后将角色移动到门附近
- 重新启用角色移动
- 播放复活动画

## 注意事项
- 确保陷阱对象有Collider2D组件
- 确保门对象有Tag为"门"
- 确保角色有Animator组件和动画控制器
- 如果使用不同的移动脚本，需要修改脚本中禁用/启用移动的代码
- 可以根据需要调整复活延迟时间和复活位置偏移量
